# Virtualmind test - backend


### Requisitos
Debe estar instalado en el ambiente de prueba:
        - Node.js
        - MongoDB
        
Al momento de seguir las instrucciones:
    - Una instancia de MongoDB (mongod.exe) debe estar en ejecución.
    
### Instrucciones
1 - Abrir una terminal en la raiz del proyecto.
2 - Ejecutar los siguientes comandos en el orden dado:
```sh
$ npm install
$ npm run initdb
$ npm start
```

La aplicación quedara servida en el puerto 3000.
